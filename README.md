# servidor-domestico-pragmatico--es-ES
Manual práctico para montar un servidor doméstico con Ubuntu Server, Docker, Immich, Navidrome y Jellyfin.

Este manual se ha escrito en español porque está pensado, ante todo, para personas hispanohablantes.

Ya existen muchos manuales excelentes en inglés, lo cuál es estupendo. Pero no todo el mundo se siente cómodo aprendiendo tecnología en otra lengua, especialmente cuando aparecen conceptos nuevos, comandos, errores y decisiones que conviene entender bien.

La intención de este proyecto es aportar una guía clara, cercana y completa en español, sin obligar al lector a traducir mentalmente cada paso mientras intenta montar su servidor.
