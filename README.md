# servidor_domestico_pragmatico_es-ES

Por Internet hay cientos de manuales de cómo montarse en casa un servidor doméstico para montarte los servicios que quieras (fotos, películas, música, contraseñas, archivos...), pero casi todos están basados en montarlos desde Windows.

Y, como yo no uso Windows hace décadas, me encontré con muy poca info de cómo hacerlo desde un mac, lo cuál no debería ser un problema dado que la terminal del mac usa zsh (y bash, y fish, y...) y Ubuntu (que es el SO que le voy a meter al servidor, pues no debería haber muchos problemas.

Así que aquí tenéis un manual práctico para montar un servidor doméstico con Ubuntu Server, Docker, Immich, Navidrome y Jellyfin.

También adelanto que el manual se ha escrito en español porque está pensado, ante todo, para personas hispanohablantes.

Ya existen muchos manuales excelentes en inglés, lo cuál es estupendo. Pero no todo el mundo se siente cómodo aprendiendo tecnología en otro idioma, especialmente cuando aparecen conceptos nuevos, comandos, errores y decisiones que convienen entender bien.

La intención de este proyecto es aportar una guía clara, cercana y completa en español, sin obligar al lector a traducir mentalmente cada paso mientras intenta montar su servidor, y hacerlo desde macOS.
